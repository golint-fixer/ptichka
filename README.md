# tgtm

Tweets go to mail

[![Build Status](https://travis-ci.org/danil/tgtm.svg)](https://travis-ci.org/danil/tgtm)

## Description

tgtm should be run by cron periodically.

tgtm get your timeline messages and send them to you by mail.
