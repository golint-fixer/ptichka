# tgtm

Tweets go to mail

## Description

tgtm should be run by cron periodically.

tgtm get your timeline messages and send them to you by mail.

## Limitations/Bugs

So if your have high flow then should process them often.

### Sort order

Sent mails can come in the wrong order.
